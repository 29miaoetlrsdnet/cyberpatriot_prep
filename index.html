Download Instructions

  1. Download both files above.
  2. Open your terminal (if it's not already open) and enter sudo -v
  3. Enter your password and authenticate.
  4. Use this code to activate the installer:
  5. cd ~/Downloads && chmod u+x installer.sh && ./installer.sh && cd
  6. You should see a message that says that installation has been completed successfully, the installer software will remove itself automatically after a successful installation.

Available Scripts

  • cp_auditor11.sh (cyberaudit) - resolves most security issues.
  • faillock.sh (faillock) - configures a lockout policy.
  • Use cyberaudit to resolve most security issues.
  • After you are done and have finished the image to the best of your ability, use faillock to configure a lockout policy.
  • This will gain you some points, but it will also lock you out of your account and break authentication.
  • Double-click a file to run it.

Instructions of Use

  1. Ensure your directory is ~/Desktop
  2. If not, run cd ~/Desktop
  3. Run cyberaudit first.
  4. Double-click cyberaudit, if it does not open, or returns an error, run this code to launch manually:
    sudo /opt/cpadu/cp_auditor11.sh
  5. You should see the following in your terminal:
====Cyberpatriot Audit and Defense Utility==== 
Look for security issues.                --(1)
Look for and repair security issues.     --(2)
View more information.                   --(3)
Advanced options.                        --(4)
Quit the program.                        --(5)

Select an option to continue:
  1. Choose the option you need:
    • 1 — Quick security check
    • 2 — Full system check and repair (recommended if unsure)
    • 3 — More information
    • 4 — Provides help with forensics quesitons and tasks covered in the readme file
  2. Follow the prompts shown in the terminal.
  3. Use y for yes and n for no.
  4. After running cyberaudit, run faillock if you wish to make a last-ditch effort to gain some points.
    • Double-click faillock, if it does not open, or returns an error, run this code to launch manually:
      sudo /opt/cpadu/faillock.sh
    • You will be asked to provide confirmation that you wish to run the program, enter y for yes and n for no.
  5. If you have any questions, feel free to ask.

Resetting Faillock Configuration

If you change your mind after using faillock.sh, run the following commands to revert to the original configuration:

sudo rm /usr/share/pam-configs/faillock
sudo rm /usr/share/pam-configs/faillock_reset
sudo rm /usr/share/pam-configs/faillock_notify
sudo sed -i '/pam_faillock.so/d' /etc/pam.d/common-auth
sudo pam-auth-update

When prompted, press Tab and then Space to exit without making changes.
